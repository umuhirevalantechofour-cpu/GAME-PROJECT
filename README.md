# UMUHIRE VALANTE CHOFOUR – Game Project

**Class:** L5 SWD

## Game Description

This is a simple browser-based game created using **HTML, CSS, and JavaScript only**.
The goal of the game is to demonstrate player control skills and overcome obstacles efficiently.

## How the Game Works

* The player uses the **keyboard** or **mouse** to control the character.
* The objective is to avoid obstacles, collect points, and reach the game’s goal.
* The game provides **feedback** on the player’s performance, score, and actions.

## Project Contents

* **index.html** – The main HTML file of the game
* **style.css** – Stylesheet controlling the game’s appearance (colors, fonts, layout)
* **script.js** – Game logic and behavior (event handling, animations)

## How to Play

1. Open **index.html** in your browser.
2. Use the **keyboard or mouse** to control the player.
3. Try to overcome obstacles and collect as many points as possible.

## Technologies Used

* **HTML5** – To build the structure of the game page
* **CSS3** – To style the game (colors, fonts, layout)
* **JavaScript** – To make the game interactive, handle events, and animate objects

## Additional Information

This game was created as a school project for the **L5 SWD class** to demonstrate knowledge in building simple web-based applications.
